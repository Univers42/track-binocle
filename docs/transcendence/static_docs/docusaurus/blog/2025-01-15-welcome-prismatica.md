Welcome to **Prismatica** — a polymorphic data platform built as part of the ft_transcendence project at 42.

<!-- truncate -->

## What is Prismatica?

Prismatica is a modern full-stack platform featuring:

- **Dual-Database Architecture** — PostgreSQL 16 for relational data, MongoDB 7 for flexible documents
- **REST API** with Express and schema-validated endpoints
- **React 19 + Vite 6** frontend with TypeScript
- **Docker Compose** orchestration for development and production
- **ABAC Security** — Attribute-Based Access Control at every layer
- **NestJS Backend** with Prisma ORM
