The infrastructure define the material and technique foundation. It designs all the physical ressources, virtuals and necessary network to run the project and host the architecture.

## Examples
physical server or cloud(AWS, GCP, Azure), database with real datas, routers, firewalls, load balancers, network cables, containers, certificates, TLS, VPN, reverse proxies, API gateways, monitoring tools, CI/CD pipelines, etc.

## The problem that they answer:
- Where the code is held ?
- What are the ressources needed to run the project ? (CPU, RAM, storage, network bandwidth, etc.)
- How to securise the reds ? (firewalls, VPN, TLS, etc.)
